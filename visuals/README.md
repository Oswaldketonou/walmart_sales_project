# 📊 Visuals

This folder contains all **model-driven** and **storytelling-ready** visualizations generated during the Walmart Sales Forecasting project. These graphics support the case study, evaluation workflow, and final business insights.

## 📈 Model Evaluation Visuals
Visuals that compare model predictions, assess error patterns, and evaluate performance:

- `best_model_actual_vs_predicted.png`
- `best_model_pred_vs_actual_scatter.png`
- `best_model_residual_distribution.png`
- `lm_pred_vs_actual.png`
- `lm_residuals.png`
- `small_models_pred_vs_actual.png`

These plots help validate model accuracy, detect bias, and compare baseline vs. Random Forest performance.

## 🌲 Feature Importance Visuals
Feature importance charts for all Random Forest model sizes:

- `rf_small1_importance.png`
- `rf_small2_importance.png`
- `rf_small3_importance.png`
- `rf_medium_importance.png`
- `rf_large_importance.png`
- `rf_all_models_importance_comparison.png`

These visuals highlight which features most strongly influence weekly sales predictions across scaling experiments.

## 🧪 Error Distribution Visuals
Visuals showing how prediction errors behave across models:

- `rf_small_error_dist.png`

These help diagnose variance, bias, and model stability.

---

All visuals in this folder are **final outputs** used in the case study, Power BI dashboard, and scenario simulation narrative.  
Exploratory Tableau visuals are stored separately in the `/tableau_eda` folder.
